A plugin for Wordpress that adds a convenient poker hand converter utility to your Wordpress post editor.

== Description ==

Convert your poker hand histories for posting in forums, chatting as plain text, or blogging with HTML!

www.handconverter.com

== Installation ==

1. Upload the thc-wordpress folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the new THC icon in your editor to paste in your raw hand history files for conversion

== Frequently Asked Questions ==

= Which poker sites are supported? =

All of the major sites are supported. If it works on www.handconverter.com, it works in this plug-in.
